## 📋 后端 API 需求

### 新增接口需求

#### 1. 轮次管理
```
GET /admin/api/rounds?page=1&pageSize=20
Response: { items: [...], total: 100 }

GET /admin/api/rounds/:id
Response: { id, roundNumber, prizePool, winner, ... }
```

#### 2. 系统日志
```
GET /admin/api/system-logs?page=1&pageSize=50&level=ERROR
Response: { items: [...], total: 500 }

GET /admin/api/system-logs/export.csv
Response: CSV file
```

#### 3. 实时监控
```
GET /admin/api/game/state
Response: { prizePool, countdown, roundNumber, lastBuyer, gameActive, ... }

GET /admin/api/game/recent-purchases?limit=20
Response: [{ id, user, amount, createdAt, ... }]
```

#### 4. 仪表盘趋势
```
GET /admin/api/dashboard/trend?days=7
Response: {
  days: [
    { date: '2026-05-25', recharged: 1000, withdrawn: 500 },
    ...
  ]
}
```
